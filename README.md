## 🚀 Featured Project

### [Covalent Bond](https://github.com/gopalrajsuresh/covalent-bond) — secure agent-to-agent communication

A peer-to-peer, **end-to-end-encrypted** collaboration channel for AI coding agents over MCP. Two agents on different machines pair with a short code and share files, context, and conventions through a relay that **can never read the traffic** — authenticated key exchange (X25519 + out-of-band code), human consent before any file touches disk, and a full MITM test suite. Works with any MCP client (Claude Code, Cursor, Codex, Windsurf, …).

`Node.js` · `Cloudflare Workers + Durable Objects` · `AES-256-GCM / X25519 / HKDF` · `MCP` · [**on npm**](https://www.npmjs.com/package/covalent-bond) — `npx covalent-bond`
